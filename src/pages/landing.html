chalkgrades is the ultimate tool for accessing student's exams in real-time.

Made with care and attention to detail, over a year, in Hungary. If you want to help contribute or just want to chat, use the options below. And for students, welcome to hell.

How does it work?

Every student has an account, and every teacher has multiple classes they test in. Teachers can make exams in workspaces, that students can start when it's public, not see when it's private, and see their grade once it's public again.

Work in progress

Chalk is still in it's somewhat early stages, so I would absolutely appriciate your contributions! The links to the documentation (useful for debugging error codes) and the GitHub repository are on the navigation bar.
